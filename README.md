# ThermOS

This is a Project taken on by me, r4dianc3, and is the brainchild of talking to myself in the bathroom mirror.

## It will include-

- ThermOS(Operating system)
- Magma(Programming language)
- Spark(Image File Format)
- Blaze(Video File Format)
- Echo(Sound File Format)
- Nitrogen(Browser)
- Firebreak(Search engine)

## Why I've created it

I decided to create this project to make an virtual ecosystem that was both completely free and open-source and also was completely usable on any computer anywhere, as long is it could display something.
